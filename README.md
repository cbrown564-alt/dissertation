# Training-Free Multi-Agent LLM System for Epilepsy Letter Extraction

This repository is being built into two project outputs:

1. A 5,000-word dissertation paper, formatted into the supplied eight-page conference template.
2. A complete end-to-end multi-agent extraction framework with an evaluation harness and reproducible supporting materials.

The first implemented task is seizure-frequency extraction, using the synthetic subset generated by Gan et al. as the initial evaluation set. The code is intentionally runnable without external LLM credentials: it provides deterministic baseline agents with the same boundaries intended for the final LLM-backed multi-agent system.

## Repository Map

- `src/epilepsy_agents/` - Python package for loading data, parsing labels, running agents, and evaluating outputs.
- `prompts/agent_roles.md` - role definitions for the future LLM-backed agents.
- `docs/` - dissertation, literature review, specification, governance, and evaluation scaffolds.
- `scripts/model_probe.py` - local hardware/runtime probe for local LLM feasibility checks.
- `tests/` - lightweight unit tests for label normalisation.
- `outputs/evaluation/` - generated evaluation outputs.

## Quick Start

Run a smoke evaluation on the synthetic subset:

```powershell
$env:PYTHONPATH = "src"
python -m epilepsy_agents.cli evaluate --data synthetic_data_subset_1500.json --limit 100 --pipeline multi
```

Compare the simpler single-pass baseline:

```powershell
$env:PYTHONPATH = "src"
python -m epilepsy_agents.cli evaluate --data synthetic_data_subset_1500.json --limit 100 --pipeline single
```

Write full evaluation artifacts:

```powershell
$env:PYTHONPATH = "src"
python -m epilepsy_agents.cli evaluate --data synthetic_data_subset_1500.json --pipeline multi --out outputs/evaluation/multi_full.json --csv outputs/evaluation/multi_full.csv
```

## Current Scope

The current framework evaluates:

- Exact structured-label match.
- Parsed monthly-rate match within 15% tolerance.
- Pragmatic class F1: `infrequent`, `frequent`, `UNK`, `NS`.
- Purist class F1 using the finer scheme described in Gan et al.
