### Current Question

How can the deterministic and future LLM-backed harnesses reduce seizure-frequency extraction errors while preserving evidence support and auditability?

### Latest Evidence

  - `h002_multi_agent_verify`: exact 0.20, monthly 0.35, pragmatic micro-F1 0.42, purist micro-F1 0.40.
  - `h001_single_pass`: exact 0.18, monthly 0.33, pragmatic micro-F1 0.40, purist micro-F1 0.38.
- Run records: [multi n100](../project_state/runs/20260424T090105Z_h002_multi_agent_verify_n100.json), [single n100](../project_state/runs/20260424T090110Z_h001_single_pass_n100.json).

### Blocker Or Risk

### Next Action

Pick one failure family, likely unknown/no-reference handling, seizure-free handling, or cluster parsing, then rerun paired 100-row smoke tests.
